## 简介

本教程系统深入地介绍污水生物处理建模方法，以 **Peterson 速率矩阵** 作为统一的数学框架。内容从微生物代谢基础出发，逐步涵盖 ASM 模型族（ASM1、ASM2d、ASM3）、反应动力学理论、COD 分馏、Python 实现及模型校准，兼顾理论严谨性与工程实用性。

### 学习收获

- Peterson 速率矩阵的数学结构与物理意义
- 从第一性原理出发推导并实现 ASM1/ASM2d/ASM3 模型
- COD 分馏方法及其在模型建立中的意义
- 生物系统中的 Monod 动力学、化学计量学与质量守恒
- 基于 Python 的活性污泥过程数值模拟
- 模型校准与灵敏度分析的实操流程
- 污水处理建模领域的研究前沿
