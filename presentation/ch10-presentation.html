GMP / MODELLING PRACTICE

模型校准
方法论

从“参数拨号”到“系统诊断”的范式转移

典型的误区

"出水氨氮高了，那就把最大生长速率 μ_A 调大一点。"

⚠️ 危害：

掩盖进水分馏错误、无法预测新工况、参数失去物理意义。

校准的本质

校准不是“调参数”，而是“诊断”数据与模型之间的不一致性。

  • 最小化预测偏差 (拟合优度)
  • 保持参数物理合理性 (可解释性)
  • 确保独立数据集验证 (泛化能力)

三大主流校准协议

BIOMATH

根特大学 8 步法。最早的规范框架，强调水力模型校准与灵敏度分析。

STOWA

荷兰应用水研究基金会。工程导向，强调“先稳态、后动态”的迭代顺序。

GMP (IWA 统一协议)

整合全球协议的权威标准。引入“校准层级”与“校准生命周期”概念。

校准层级 (Hierarchy)

L1

仅调整进水分馏

最常用，解决 70% 的偏差。不修改动力学参数。

L2

分馏 + 关键动力学参数

调整灵敏度最高的参数 (如 μ_A)。适用于特殊水质。

L3

全参数优化

需要大量独立实验 (OUR 呼吸测定、硝化速率测定)。

进水分馏：校准的核心

进水分馏决定了模型预测的准确性。S_I (溶解惰性) 直接决定出水 COD 下限。

推荐方法：OUR 呼吸测定法

通过 OUR 随时间的变化曲线，分离 S_S (快速峰) 与 X_S (缓慢水解段)。

典型分馏 (市政污水):

S_S (易降解): 15%
X_S (慢速): 45%
X_I (颗粒惰性): 32%
S_I (溶解惰性): 8%

水力模型校准 (RTD)

真实混合模式介于 CSTR 与 PFR 之间。

示踪剂测试 (Tracer Test):

  • 注入脉冲示踪剂 (LiCl, 荧光素)
  • 监测出水 RTD 曲线
  • 拟合 N 个 CSTR 串联模型

N 的工程估算：

  • 完全混合氧化沟: N = 1-2
  • 廊道式曝气池: N = 4-6
  • 推流式曝气池: N ≥ 8

稳态校准顺序 (不可打乱)

01 COD 守恒检查 (误差 < 5%)
02 污泥产量校准 (Y_H, f_P, X_I)
03 氧消耗 (OUR) 校准
04 硝化性能校准 (μ_A, b_A, K_NH)
05 反硝化与除磷校准

提示：底层不稳，高层必塌。严禁直接跳到硝化参数调整。

灵敏度分析：拨动哪个旋钮？

局部灵敏度指数 (S_ij): 参数变化 1% 引起输出变化的百分比。

对出水 S_NH (氨氮) 的参数灵敏度：

μ_A
- 0.92
K_NH
+ 0.65
b_A
+ 0.28
μ_H
+ 0.04

结论：μ_A 和 K_NH 是硝化过程的“主控旋钮”，而异养菌参数影响极微。

总结：校准的五个金律

01 数据质量第一，模型第二

02 进水分馏解决 70% 的问题

03 遵循非对称、不可逆的校准顺序

04 保持参数的物理可解释性

05 永远用独立数据集进行验证
