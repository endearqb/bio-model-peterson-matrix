Peterson 速率矩阵

数学结构与物理意义

一种为复杂生物系统而生的数学语言

历史来源：统一的框架

在 ASM1 (1987) 诞生之前，各研究者表达模型的方式混乱不堪：

  • 方程组形式各异
  • 组分定义不统一
  • 化学计量与动力学混杂

Peterson (1965) 引入矩阵符号，Gujer 将其引入水处理领域。

Core Value:

将复杂的微分方程组转化为模块化、可视化的表格。使模型易于比较、交流与编程实现。

矩阵的三要素

过程 \ 组分 C₁ C₂ 速率 ρ
过程 1 ν₁₁ ν₁₂ ρ₁
过程 2 ν₂₁ ν₂₂ ρ₂
  • 组分 (Columns): 所有追踪的浓度 (S_S, X_BH, S_O...)
  • 过程 (Rows): 所有生物/化学转化 (生长、衰减...)
  • 系数 (Elements): νᵢⱼ 表示每生产 1 单位参考组分，组分 j 的变化量 (+ 生成, - 消耗)

核心控制方程 (CSTR)

利用矩阵可以系统地写出任意组分的动态方程：

dCⱼ/dt = (1/HRT) · (Cⱼ,in - Cⱼ) + Σᵢ(νᵢⱼ · ρᵢ)
对流项 (Transport)
进水中组分带来的变化
反应项 (Reaction)
矩阵行与列的乘积和 (源/汇)

COD 守恒：黄金检验

由于 COD 是能量守恒量，每个生物过程必须满足：

Σⱼ ( νᵢⱼ × COD_equiv(j) ) = 0
组分COD 当量
有机物+1.0
溶解氧 S_O-1.0
硝酸盐 S_NO-2.86
氨氮 / 碱度0

物理意义：
消耗的底物能量 = 变成细胞的能量 + 传递给电子受体的能量。凡是通过守恒检验的过程，模型即具备内在自洽性。

开关函数：环境自动切换

无需 if-else 逻辑，速率方程即可自动处理好氧/缺氧切换：

正向激活 (Monod)

S / (K + S)

S 充足 → 1 (运行)
S 匮乏 → 0 (停止)

负向抑制

K / (K + S)

S 存在 → 0 (抑制)
S 消失 → 1 (解除)

本章小结

  • 核心工具： Peterson 矩阵是描述复杂反应动力学的工业标准。
  • 三个要素： 过程行、组分列、系数。
  • 正交分离： 实现了化学计量与动力学的独立模块化。
  • 守恒约束： COD 守恒是验证矩阵自洽性的唯一真理。
