01 / Literature Core Textbooks

必读核心教材

从源头掌握模型理论与工程应用的权威著作：

I. 第一梯队：模型原著

  • 《Activated Sludge Models...》 (IWA, 2000)
    ★★★★★ 所有模型的权威原始文献与参数集。
  • 《Biological Wastewater Treatment》 (Henze等, 2008)
    ★★★★★ 最全面的生物处理理论教材，研究生首选。

II. 第二梯队：工程应用

  • 《Guidelines for Using ASM》 (IWA, 2012)
    第一个真正意义上的“模型用户手册”（GMP 协议）。
  • 《Wastewater Engineering (5th)》 (Metcalf & Eddy, 2014)
    工程实践百科全书，含大量设计计算表格。
  • 《Systems Analysis for Water Tech》 (Gujer, 2008)
    Peterson 矩阵引入者的亲笔教材，数学推导严密。

03 / Tools Software & Code

开源工具与商业软件

从理论验证到全厂数字孪生的落地工具栈：

Python 生态 (开源)

  • WaterTAP： 美国 DOE 开发，支持 ASM1/2d 及最优化设计。
  • QSDsan： UIUC 开发，支持 LCA 及 BSM1 全厂基准模型。
  • pyASM： 轻量级 ASM 实现，适合教学。
  • 自学起点： 本教程第 9 章提供的 300 行 Python 代码。

商业与学术软件

  • Matlab/Simulink： 获取 IWA 官方免费 BSM1 测试框架。
  • BioWin (EnviroSim)： 业界主流，可申请 30 天教育试用。
  • GPS-X (Hydromantis)： 强大的图形化全厂模拟软件。
  • WEST (DHI) & SUMO： 均提供学术版或演示版。
04 / Action Plan 16-Week Roadmap

16 周系统学习计划

阶段一：打地基 (第 1-4 周) 掌握历史背景、微生物代谢、COD 分馏，能手算单底物 CSTR 稳态（出水、污泥产量、OUR）。
阶段二：核心突破 (第 5-8 周) 深入 Peterson 矩阵，推导 ASM1 系数验证 COD 守恒，对比理解 ASM2d 除磷机制与 ASM3。
阶段三：编程实战 (第 9-12 周) 运行 Python ASM1 模拟，完成灵敏度分析；阅读 GMP 协议学习模型校准方法论。
阶段四：进阶与前沿 (第 13-16 周) 接触 BioWin/GPS-X，精读数字孪生与机器学习前沿论文；执行一个完整的假设污水厂项目。
05 / Strategy Time Management

学习精力分配策略

建议每周投入 8-10 小时，注重主动回顾与多维度验证：

理论学习 (3h)
推导练习 (2h)
Python 编程 (2h)
文献阅读 (2h)
复习 (1h)
  • 主动回顾 (Active Recall)： 抛开书本，在白纸上重推 Peterson 矩阵。
  • 交叉验证： 对比 Python 手写代码与商业软件的模拟结果，深挖差异。
  • 真实数据： 从污水厂年报中寻找真实数据练习进水分馏。

07 / Epilogue The Engineer's Second Language

建模是工程师的"第二语言"

"模型是理解的工具，而不是目的本身。"

当你能够独立构建一个 Peterson 矩阵、验证 COD 守恒、并用 Python 模拟动态响应时，你就已经从 “使用者” 成长为了 “理解者”。这是所有后续进阶（数字孪生、ML 混合建模、节能减排）的共同基础。

学无止境，但方向明确。祝学习顺利！
